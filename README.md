# 🌐 GitHub 局部翻译提示插件

## 📌 项目简介

这是一个专为 GitHub 页面设计的浏览器插件，用于在不改变原始英文内容的前提下，为关键文本添加中文提示。

插件会自动识别 GitHub 页面中的常用功能区域（如导航栏、按钮等），并在原文本下方以“小字注释”的形式展示对应中文说明，帮助用户更轻松地理解页面内容。

---

## ✨ 产品特点

* 🎯 **仅作用于 GitHub 页面**
* 🪶 **非侵入式翻译（保留原文）**
* ⚡ **无需联网，无需 AI，完全本地运行**
* 🔄 **自动适配 GitHub 动态页面**
* 🌐 **支持多语言切换（可扩展）**
* 🎛️ **提供开关控制与语言选择界面**

---

## 🖼️ 使用效果

在 GitHub 页面中，例如：

Issues   问题
↓
Issues   问题
问题（Issue 跟踪）

原文保持不变，下方增加中文提示，阅读更加直观。

---

## ⚙️ 功能说明

### ✅ 翻译开关

可通过插件面板随时开启或关闭翻译功能

### ✅ 多语言支持

可在插件中选择不同语言（基于本地语言文件）

### ✅ 自动生效

进入 GitHub 页面后自动运行，无需额外操作

---

## 🚀 安装与部署教程

### 📁 第一步：下载项目

将本项目下载到本地，并解压（或直接 git clone）

---

### 🌐 第二步：打开浏览器扩展页面

在 Chrome 或 Edge 浏览器地址栏输入：

chrome://extensions/

（Edge 用户使用：edge://extensions/）

---

### 🔧 第三步：开启开发者模式

在页面右上角打开：

“开发者模式（Developer Mode）”

---

### 📦 第四步：加载插件

点击：

“加载已解压的扩展程序（Load unpacked）”

然后选择本项目文件夹

---

### ✅ 第五步：安装完成

插件加载成功后，会出现在浏览器扩展列表中

---

### 🧪 第六步：开始使用

1. 打开 GitHub 任意页面
2. 进入任意仓库
3. 页面中的关键英文内容下方会自动显示中文提示

---

## 🔄 更新插件

当你修改了插件内容后：

1. 打开扩展页面（chrome://extensions/）
2. 找到本插件
3. 点击“刷新”按钮
4. 刷新 GitHub 页面

---

## ⚠️ 注意事项

* 插件仅在 GitHub 页面生效
* 首次安装后请刷新页面
* 修改语言或设置后，可能需要刷新页面
